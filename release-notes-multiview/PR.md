Add Style Builder, layout reorder/backup, and dashboard session persistence (v0.4.0)

This release adds a visual Style Builder for creating custom multiview layouts (static tiles, dynamic rows/grids, background images, live preview), plus drag-to-reorder layouts and JSON backup/restore in the dashboard. Layouts now use stable internal IDs instead of position numbers so reordering or deleting one no longer changes another's stream URL or swaps a viewer's stream on reconnect; the M3U tvg-id, stream URL, and EPG channel id format changed from `multiview_N` to `mv-<id>` as part of this, so existing multiview channels in Dispatcharr will likely be re-created (not renamed) on the next M3U/EPG refresh, and channel numbering, groups, and manual EPG mapping should be re-checked afterward. The dashboard also gains a configurable mount path, plugin/Dispatcharr version display, silent session refresh with remembered username, and a searchable channel picker. Fixes include a layout-count desync that could delete existing multiviews, audio desync on unstable streams, small tile-gap rendering issues in Auto Row and Featured layouts, Style Builder guide-line and aspect-lock bugs, and an M3U refresh that was silently skipped when EPG refresh failed.

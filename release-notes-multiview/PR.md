Fix settings/dashboard crash, dashboard connection leak, and Auto Grid row gap (3.0.1)

Fixes a crash where selecting the same channel in more than one slot of a layout with EPG forwarding enabled would break both the plugin's settings page and the web dashboard with a "Duplicate options" error from the underlying UI library. Also fixes a database connection leak in the plugin's own dashboard server that could build up over time and cause the dashboard, and sometimes the settings page, to stop responding until Dispatcharr was restarted. Finally, the Auto Grid layout now avoids doubling up its letterboxing gap between stacked rows, pushing padding to the outer top and bottom edges instead so rows meet cleanly in the middle.

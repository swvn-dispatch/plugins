Add web dashboard, PyAV auto-reinstall, and stream stability fixes (v0.3.0)

Adds a mobile-friendly PWA dashboard (served at `/dash/`, off by default) for editing plugin settings and managing active streams without the Dispatcharr admin UI. PyAV now auto-reinstalls itself once the user has consented once, so a vendored copy that goes missing or outdated no longer requires a manual re-click. The settings page warns when Dispatcharr's Channel Initialization Timeout is set too low, and settings fields have been reorganized into Global and Video sections for clarity. Also fixes the seam gap between stacked tiles in the Featured layout, audio drift on unstable streams, and reverse-proxy access to the dashboard API.

Replace debug dashboard with login-gated PWA dashboard, add settings editing

Replaces the old debug-only dashboard with a mobile-friendly PWA served under a configurable mount path, gated behind Dispatcharr login, that lets users view live channel/client and media server status and edit plugin settings without the Dispatcharr admin UI. Dashboard-related settings were renamed (Debug Server -> Dashboard) and grouped under clearer headers. Also fixes the dashboard reliably starting in the same worker process as the stream monitor, with automatic self-healing retry if the first startup attempt fails, and reduces log noise from repeated shared-identifier messages.
